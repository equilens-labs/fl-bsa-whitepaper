
\section{Compliance Mapping}
We map framework requirements (\emph{e.g.}\ \ECOA/\RegB, SR 11-7, \EUAIAct) to controls and evidence artefacts. The detailed mapping for this run is given in Appendix~D. This paper documents model purpose, limitations, monitoring, provenance, and evidence-chain artefacts that reviewers can map to SR 11-7 \citep{sr11_7} expectations.
