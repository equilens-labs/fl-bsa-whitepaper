# Legacy Pre-v5 Intake Files

This directory preserves historical intake material that is not part of the current stable
`v5.0.0` whitepaper intake surface.

The archived files include predecessor provenance manifests, the old `manifest_gate_wp.json`,
2025-era summary CSVs, `runs.json`, and `claims_to_substantiate.md`. They are retained for
traceability only.
Do not cite them as the current evidence inventory, current claim list, current certificate-signing
posture, or current intake basis.

For the current intake surface, use the files directly under `intake/`, especially
`intake/manifest.json`, `intake/pack_intent.json`, `intake/metrics_long.csv`,
`intake/metrics_uncertainty.json`, `intake/regulatory_matrix.csv`, and the unsigned certificates in
`intake/certificates/`.
